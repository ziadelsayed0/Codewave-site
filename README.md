# Hundeling Your Notes

## Technologies

### Backend
<ul>
  <li>Django</li>
  <li>JWT</li>
  <li>SQLite</li>
</ul>

### Frontend
<ul>
  <li>React</li>
  <li>Tailwind</li>
  <li>JWT-Decode</li>
</ul>

## Setup Backend
```
pip install -r requirements.txt
```

### Compiles for development (Backend)
```
python manage.py migrate
python manage.py runserver
```

## Setup Frontend
```
npm install
```

### Compiles for development (Frontend)
```
npm run build
npm run dev
```
